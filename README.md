# Khumeia Café - Portal de Especialidad
Proyecto de e-commerce y menú digital para Khumeia Café.

## Stack Tecnológico
- **Frontend:** Next.js + Tailwind CSS
- **Backend:** Supabase (PostgreSQL)
- **Hosting:** Vercel

## Configuración
Para correr este proyecto, necesitas configurar las variables de entorno de Supabase en un archivo `.env`.
